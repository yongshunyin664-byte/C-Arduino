# 第2周实操题

## 一、C 语言实操题

1. 请编写一段 C 代码，使用 `if...else` 判断一个整数是否为偶数，并输出“偶数”或“奇数”。

2. 请编写一段 C 代码，使用 `switch...case` 判断星期编号（1-7），并输出对应的星期名称，例如 1 对应“星期一”。

3. 请编写一段 C 代码，使用 `for` 循环输出 1 到 10 的所有整数，并在遇到 5 时使用 `break` 结束循环。

4. 请编写一段 C 代码，使用 `while` 循环计算 1 到 100 的所有整数之和。

5. 请编写一段 C 代码，使用嵌套循环输出以下矩阵：

```
1 1 1
2 2 2
3 3 3
```

## 二、Arduino 实操题

1. 请设计一个 Arduino 程序，读取按钮输入并控制一个 LED：按钮按下时 LED 亮，松开时 LED 熄灭。

2. 请设计一个 Arduino 程序，使用 `analogWrite()` 使 LED 逐渐从暗到亮再从亮到暗循环。

3. 请设计一个 Arduino 程序，读取电位器 `analogRead()` 的值，并将其转换成 0-255 的 PWM 输出，用于控制 LED 亮度。

4. 请设计一个 Arduino 程序，使用蜂鸣器 `tone()` 发出 1kHz 的声音 0.5 秒，再停止 0.5 秒，循环执行。

5. 请说明在 Arduino 实验中，如何防止按钮控制 LED 时出现“抖动”误判？
