### 题目2：计算圆的面积

已知圆的半径 `r = 5.0`，圆周率 `PI = 3.14159`，求圆的面积。

**公式**：面积 = PI × r × r

**要求**：
- 用 `#define` 定义 PI
- 半径用 `double` 类型
- 用 `printf` 输出结果，格式为：`圆的面积是：XX`

```c
// 请在此处写出你的代码：
#include <stdio.h>

#define PI = 3.14;

int main(void)
{
    double r = 5;
    double area = PI*r*r;
    printf("圆的面积是：%f\n",area);

    return 0;
}
```

### 题目3：简单计算器

**要求**：
- 用 `scanf` 读取两个整数
- 用 5 个算术运算符分别计算
- 每行输出一个结果

**运行示例**：
```
请输入两个整数：10 3
和：13

余数：1
```

    printf("积:%d\n",a*b);
    printf("商:%d\n",a/b);
    printf("余数:%d",a%b);

### 题目4：温度转换

输入一个摄氏温度，转换成华氏温度并输出。

**公式**：华氏度 = 摄氏度 × 9 / 5 + 32

**要求**：
- 温度用 `float` 类型
- 用 `scanf` 读取输入
- 用 `printf` 的 `%f` 输出结果

```c
// 请在此处写出你的代码：
#include <stdio.h>

int main(void)
{

    float s,h;
    printf("输出一个摄氏温度:");
    scanf("%f",&s);
    h = s * 9 / 5 + 32;
    printf("华氏摄氏为:%f\n",h);

    return 0;

}
```
