### 四、简答题（要点）

**86.** 三种方法：①先声明后赋值；②声明时初始化（推荐）；③同时声明多个同类型变量。推荐第②种，因为能避免未初始化变量包含垃圾值导致的不可预期行为。

**87.** 区别：①处理阶段——`#define`在预处理阶段，`const`在编译阶段；②类型检查——`#define`无，`const`有；③占用内存——`#define`不占（直接替换），`const`占内存。

**88.** `char`(%c)、`short`(%hd)、`int`(%d)、`long`(%ld)、`long long`(%lld)、`float`(%f)、`double`(%f输出/%lf输入)。

**89.** 输出：
```
13
3
1
```

**90.** 错误：① `1st_num` 数字开头，非法变量名；② `float` 是关键字，不能作变量名；③ `scanf("%d", a)` 缺少 `&`，应为 `scanf("%d", &a)`。

**91.** `char ch = 'B';`

**92.** `scanf("%lf", &变量名);` 例如 `scanf("%lf", &d);`

**93.** `printf("%lld", 变量名);` 例如 `printf("%lld", ll);`

**94.** `scanf("%hd", &变量名);` 例如 `scanf("%hd", &s);`

**95.** `#define PI 3.14`

**96.** 输出：`16`（N=5, x=5+3=8, x=8*2=16）

**97.**
```c
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    printf("%d\n", n);
    return 0;
}
```

**98.** 输出：
```
A

```
（因为 `%c` 输出字符 A，`%d` 输出其 ASCII 码 65）

**99.** "静态变量生命周期是整个程序"指程序运行期间静态变量一直存在（不会像局部变量那样函数结束就销毁），但程序结束后内存全部回收。这和"程序结束后数据还在"是两回事——程序结束后内存数据全部消失，想要数据持久化必须写入硬盘（文件）。

**100.**
```c
#include <stdio.h>
int main() {
    int n;
    float f;
    scanf("%d %f", &n, &f);
    printf("%d\n", n * 2);
    printf("%f\n", f / 2);
    return 0;
}
```
