## 4. 纠错题

21. 请指出并改正下列代码中的错误：

```

```

```

```

```

23. 请指出并改正下列代码中的错误：
```c
printf("%d", 3.14);
```

答题框：
```
printf("%f", 3.14);
```

24. 请指出并改正下列代码中的错误：
```c
int a = 5;
int b = 2;
printf("%d", a / b);
```

答题框：
```
【解析】这段代码的关键点是“整数除法”。
因为 `a` 和 `b` 都是 `int` 类型，所以 `a / b` 结果也是整数，`5 / 2` 会得到 `2`，不是 `2.5`。
所以这段代码本身没有语法错误，重点是要知道：整数除法会丢掉小数部分。
如果想得到小数结果，需要把其中一个数改成浮点数，例如：
```c
printf("%.2f", 5.0 / 2);
```
```

25. 请指出并改正下列代码中的错误：
```c
#define PI 3.14
printf("%d", PI);
```

答题框：
```
【解析】这里的错误是“格式控制符不匹配”。
`PI` 的值是 `3.14`，属于浮点数，应该用 `%f` 输出，而不是 `%d`。
正确写法是：
```c
#define PI 3.14
printf("%f", PI);
```
```
