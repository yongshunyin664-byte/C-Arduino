# Arduino UNO 基础元件笔记

## 1. Arduino UNO 是什么
- Arduino UNO 是一块常用的微控制器开发板。
- 它相当于一个“可编程的小控制器”。
- 可以用来控制 LED、蜂鸣器、传感器等元件。

## 2. 面包板是什么
- 面包板是用来搭建电路的工具。
- 不需要焊接，方便测试和调试。
- 里面有很多小孔，可以把元件和导线插进去。

## 3. 杜邦线是什么
- 杜邦线是用来连接电路元件的导线。
- 它可以把 Arduino 和面包板、LED、电阻连接起来。
- 常见用途是“做电路连接”。

## 4. LED 是什么
- LED 是发光二极管。
- 通电后会发光。
- 它有正负极之分，长脚一般是正极。

## 5. 电阻是什么
- 电阻是用来限制电流的元件。
- 它可以保护 LED，防止 LED 被烧坏。
- 常见电阻值有 220Ω、330Ω、1kΩ 等。

## 6. 它们之间的关系
- Arduino UNO 负责控制。
- 杜邦线负责连接。
- 面包板负责搭建电路。
- 电阻负责保护 LED。
- LED 负责发光。

## 7. 最简单的理解
- Arduino UNO：大脑
- 面包板：实验平台
- 杜邦线：连接线
- LED：发光灯
- 电阻：保护灯的元件
