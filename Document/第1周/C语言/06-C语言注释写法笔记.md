# C语言注释写法笔记

## 1. 什么是注释
- 注释是给代码加上的说明文字。
- 程序运行时不会执行注释内容。

## 2. 单行注释
- 用 `//` 开头。
- 只能写一行。

```c
#include <stdio.h>

int main(void)
{
    // 这是一个单行注释
    int age = 18;
    return 0;
}
```

## 3. 多行注释
- 用 `/* ... */` 包起来。
- 可以写多行。

```c
#include <stdio.h>

int main(void)
{
    /*
    这是一个多行注释
    可以写很多行
    */
    int age = 18;
    return 0;
}
```

## 4. 记忆口诀
- `//`：单行注释
- `/* */`：多行注释
