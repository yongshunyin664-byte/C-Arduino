# C语言基础数据类型笔记

## 1. 整数类型

### 1.1 `short` — 短整型
- 占用 2 字节，范围约 **-32768 ~ 32767**。
- 适合存储较小的整数，节省内存。
- 语法：

```c
short s = 100;
```

### 1.2 `int` — 整型（最常用）
- 占用 4 字节，范围约 **-21亿 ~ 21亿**。
- 日常编程中用得最多的整数类型。
- 语法：

```c
int age = 18;
```

### 1.3 `long` — 长整型
- 占用 4 或 8 字节（取决于平台）。
- 适合存储较大的整数，字面量建议加 `L` 后缀。
- 语法：

```c
long l = 100000L;
```

### 1.4 `long long` — 长长整型
- 占用 8 字节，范围约 **-9×10¹⁸ ~ 9×10¹⁸**。
- 适合存储超大整数，字面量建议加 `LL` 后缀。
- 语法：

```c
long long ll = 9999999999LL;
```

### 整数类型对比

| 类型       | 占用字节 | 大致范围                          |
|------------|----------|----------------------------------|
| `short`    | 2        | -32,768 ~ 32,767                 |
| `int`      | 4        | -2,147,483,648 ~ 2,147,483,647   |
| `long`     | 4 或 8   | 视平台而定                        |
| `long long`| 8        | -9×10¹⁸ ~ 9×10¹⁸                 |

---

## 2. 浮点类型（小数）

### 2.1 `float` — 单精度浮点型
- 占用 4 字节，约 6~7 位有效数字。
- 字面量**必须**加 `f` 后缀，否则编译器当作 `double`。
- 语法：

```c
float price = 3.5f;
```

### 2.2 `double` — 双精度浮点型
- 占用 8 字节，约 15~16 位有效数字。
- 精度更高，是实际开发中最常用的小数类型。
- 语法：

```c
double pi = 3.14159265358979;
```

### 浮点类型对比

| 类型     | 占用字节 | 有效数字     | 后缀 |
|----------|----------|-------------|------|
| `float`  | 4        | 6~7 位      | `f`  |
| `double` | 8        | 15~16 位    | 无   |

---

## 3. `char` — 字符类型

- 占用 1 字节，存储一个字符（ASCII 码）。
- 用**单引号**包裹，只能放一个字符。
- 底层存的其实是整数（ASCII 编码值）。
- 语法：

```c
char c = 'A';        // 字符
char ch = 65;        // 等价于 'A'（ASCII 码 65 = 'A'）
```

---

## 4. 各类型区别总结

| 分类       | 类型                                    | 用途        |
|------------|----------------------------------------|-------------|
| 整数       | `short` / `int` / `long` / `long long` | 存整数       |
| 浮点（小数）| `float` / `double`                     | 存小数       |
| 字符       | `char`                                 | 存单个字符    |

## 5. 完整示例

### 5.1 输出示例（`printf`）

```c
#include <stdio.h>

int main(void)
{
    char ch = 'A';
    short s = 100;
    int a = 10;
    long l = 100000L;
    long long ll = 9999999999LL;
    float b = 3.14f;
    double c = 3.14159265358979;

    printf("char:      %c\n",  ch);
    printf("short:     %hd\n", s);
    printf("int:       %d\n",  a);
    printf("long:      %ld\n", l);
    printf("long long: %lld\n", ll);
    printf("float:     %f\n",  b);
    printf("double:    %f\n",  c);

    return 0;
}
```

### 5.2 输入示例（`scanf`）

```c
#include <stdio.h>

int main(void)
{
    char ch;
    short s;
    int a;
    long l;
    long long ll;
    float b;
    double c;

    // 逐个输入
    printf("请输入一个字符：");
    scanf("%c", &ch);

    printf("请输入一个短整数：");
    scanf("%hd", &s);

    printf("请输入一个整数：");
    scanf("%d", &a);

    printf("请输入一个长整数：");
    scanf("%ld", &l);

    printf("请输入一个长长整数：");
    scanf("%lld", &ll);

    printf("请输入一个浮点数：");
    scanf("%f", &b);

    printf("请输入一个双精度浮点数：");
    scanf("%lf", &c);   // 注意：double 必须用 %lf！

    // 输出验证
    printf("\n===== 你输入的是 =====\n");
    printf("char:      %c\n",  ch);
    printf("short:     %hd\n", s);
    printf("int:       %d\n",  a);
    printf("long:      %ld\n", l);
    printf("long long: %lld\n", ll);
    printf("float:     %f\n",  b);
    printf("double:    %f\n",  c);

    return 0;
}
```

## 6. 记忆口诀

- `char` — 记字符
- `short` — 短整数
- `int` — 最常用整数
- `long` — 长整数（加 `L`）
- `long long` — 超长整数（加 `LL`）
- `float` — 单精度小数（加 `f`）
- `double` — 双精度小数（默认）
