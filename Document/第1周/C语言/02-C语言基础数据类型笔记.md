# C语言基础数据类型笔记

## 1. int：整数类型
- 用来存储整数。
- 例如：10、-3、0。
- 语法：

```c
int age = 18;
```

## 2. float / double：小数类型
- 用来存储小数。
- float：单精度浮点数，精度较低。
- double：双精度浮点数，精度更高，使用更常见。
- 语法：

```c
float price = 3.5f;
double pi = 3.1415926;
```

## 3. char：字符类型
- 用来存储一个字符。
- 只能存放一个字符，通常用单引号。
- 语法：

```c
char c = 'A';
```

## 4. 三者的区别
- int：存整数
- float/double：存小数
- char：存字符

## 5. 一个简单例子

```c
#include <stdio.h>

int main(void)
{
    int a = 10;
    float b = 3.14f;
    double c = 3.1415926;
    char d = 'C';

    printf("%d\n", a);
    printf("%f\n", b);
    printf("%lf\n", c);
    printf("%c\n", d);

    return 0;
}
```

## 6. 记忆口诀
- int 记整数
- float/double 记小数
- char 记字符
