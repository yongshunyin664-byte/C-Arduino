## 一、什么是变量?

变量是程序中用于存储数据的**命名内存空间**。
每个变量在内存中占据一块空间，变量名就是这块空间的别名。
通过变量名，我们可以方便地读写内存中的数据。
我们可以把它想象成一个带标签的盒子——标签是变量名，盒子里装的是数据。

```c
int age = 25;      // age 是变量名，25 是存在里面的值
```

## 二、变量的声明与初始化(3种方法)

> **建议**：始终在声明变量时进行初始化。未初始化的变量包含随机值（垃圾值），使用前如不赋值可能导致不可预期的行为。

```c
// 1. 先声明，后赋值
int count;
count = 10;

// 2. 声明的同时初始化（推荐做法）
int count = 10;

// 3. 同时声明多个同类型变量
int a, b, c;
int x = 1, y = 2, z = 3;
```

## 三、变量命名规则（必须遵守）

### 3.1 合法规则

1. **只能由字母（A-Z, a-z）、数字（0-9）和下划线（`_`）组成**
2. **第一个字符不能是数字**，必须是字母或下划线
3. **严格区分大小写**——`age`、`Age`、`AGE` 是三个互不相关的变量
4. **不能使用C语言的关键字（保留字）作为变量名**

### 3.2 示例

```c
// ✅ 合法的变量名
int age;
int _count;
int player1;
int total_score;
int MAX_SIZE;
int Über;           // C99起支持Unicode字符（但不推荐）

// ❌ 非法的变量名
int 1st_place;      // 数字开头
int my-variable;     // 包含连字符 -
int float;           // float 是关键字
int user name;       // 包含空格
int $price;          // 包含美元符号 $
int user@email;      // 包含 @ 符号
```

## 四、变量的作用域与生命周期

```c
#include <stdio.h>

int global_var = 100;           // 全局变量——整个文件可用，程序结束后销毁

static int file_var = 50;       // 文件作用域——仅本文件内可用

int main() {
    int local_var = 10;         // 局部变量——仅在 main() 内可用
    static int static_var = 5;  // 静态局部变量——值在函数调用间保持

    {
        int block_var = 99;     // 块作用域——仅在该 {} 块内可用
        printf("%d\n", block_var);
    }
    // block_var 在此处不可访问

    return 0;
}
```

| 存储类别          | 作用域    | 生命周期    | 默认初始值 |
|------------------|-----------|------------|-----------|
| 局部变量 `auto`   | 函数/块内 | 函数调用期间 | 垃圾值    |
| 全局变量          | 整个程序  | 整个程序     | 0        |
| 静态局部 `static` | 函数内    | 整个程序     | 0        |
| 静态全局 `static` | 本文件    | 整个程序     | 0        |
| `register`       | 函数内    | 函数调用期间  | 垃圾值   |

那"静态变量生命周期是整个程序"是什么意思？
这里容易混淆，关键在于 "生命周期"说的是程序运行期间，不是程序结束之后：


                    程序运行期间
                    ├───────────────────┤
    局部变量：       诞生→→→→→→销毁        （函数结束就死）
    全局变量：       诞生→→→→→→→→→→→→销毁  （程序结束才死）
    静态变量：       诞生→→→→→→→→→→→→销毁  （程序结束才死）
它们的区别只在程序运行的时候谁先死谁后死。程序一旦结束，全都没了。

真正能"留下来"的只有硬盘上的东西

内存（RAM）： 程序运行时存数据  →  程序结束 → 全部清空
硬盘（磁盘）：文件、数据库      →  程序结束 → 依然存在
所以如果你想数据在程序结束后还在，就写到文件里去。内存里的东西，没有任何例外，全是一次性的。
因为变量是程序中用于存储数据的**命名内存空间**，
每个变量在内存中占据一块空间，变量名就是这块空间的别名。(知识点来自"一、什么是变量?")

## 五、什么是常量、 #define是什么？

### 5.1 什么是常量？

常量是程序中**固定不变**的值。一旦定义，在程序运行期间不可修改。

C语言中有两种方式定义常量：

| 方式 | 关键字 | 特点 |
|------|--------|------|
| 宏定义 | `#define` | 预处理阶段文本替换，无类型检查，不占内存 |
| const常量 | `const` | 有类型检查，占内存，本质是只读变量 |

### 5.2 `#define` 宏定义

```c
#include <stdio.h>

#define PI 3.1415926
#define MAX_STUDENTS 100
#define SCHOOL_NAME "育才中学"

int main() {
    printf("圆周率：%f\n", PI);
    printf("最大学生数：%d\n", MAX_STUDENTS);
    printf("学校名称：%s\n", SCHOOL_NAME);

    // PI = 3.14;        // ❌ 错误！不能修改 #define 常量

    return 0;
}
```

**注意：**
- `#define` 末尾**不要加分号** `;`
- `#define` 是预处理指令，在编译前做**纯文本替换**
- 习惯上宏名用**全大写**命名

### 5.3 `const` 常量

```c
#include <stdio.h>

int main() {
    const double PI = 3.1415926;
    const int MAX = 100;

    printf("PI = %f\n", PI);
    printf("MAX = %d\n", MAX);

    // PI = 3.0;          // ❌ 错误！const 常量不可修改

    return 0;
}
```

### 5.4 `#define` vs `const` 对比

| 对比项 | `#define` | `const` |
|--------|-----------|---------|
| 处理阶段 | 预处理（编译前文本替换）| 编译阶段 |
| 类型检查 | 无 | 有 |
| 占用内存 | 不占（直接替换到代码中）| 占内存（是变量） |
| sizeof 取值 | 不可以 | 可以 |
| 作用域控制 | 无（从定义处到文件末尾）| 有（受 `{}` 限制） |
| 推荐场景 | 纯数值常量、条件编译 | 需要类型安全时 |

### 5.5 记忆口诀

- `#define` — 预处理替换，无类型，大写命名，末尾无分号
- `const` — 只读变量，有类型，受作用域限制

## 八、C语言 scanf 输入笔记

### 8.1 scanf 是什么
- scanf 用来从键盘读取输入。
- 它和 printf 相反：
  - printf：输出
  - scanf：输入

### 8.2 基本用法
```c
#include <stdio.h>

int main(void)
{
    int age;
    scanf("%d", &age);
    printf("%d\n", age);
    return 0;
}
```

### 8.3 它从哪里读取
- scanf 读取的是键盘输入。
- 也就是说，你在运行程序时，在控制台里输入的内容，会被 scanf 读取。
- 比如这段

```c
int age;
scanf("%d", &age);
```
- 意思是：
- 程序会等待你在键盘上输入一个整数
- 输入完成后，这个整数就会被存到 age 里面
- 例如你在运行程序时输入：

```c
18
```

- 那么 age 就等于 18。
- 可以把它理解成：
- printf 是"把东西输出出去"
- scanf 是"从外面接收输入"

### 8.4 记住一个重点
- 读取数据时，通常要写 `&变量名`。
- 例如：`scanf("%d", &age);`

### 8.5 记忆口诀
- scanf 读输入
- printf 打输出
- `%d` 读整数
- `%f` 读小数
- `%c` 读字符

## 十、命名风格（惯例/建议）

虽然不是编译器强制的，但良好的命名风格能极大提升代码可读性和可维护性。

### 10.1 常见命名风格

| 风格                           | 示例                                    | 常用场景              |
|--------------------------------|-----------------------------------------|----------------------|
| **蛇形命名** (snake_case)      | `user_name`, `max_count`, `total_score` | C语言中最常见、最推荐  |
| **驼峰命名** (camelCase)       | `userName`, `maxCount`, `totalScore`    | 部分C项目使用         |
| **帕斯卡命名** (PascalCase)    | `UserName`, `MaxCount`                  | 结构体/类型名常用     |
| **全大写下划线** (UPPER_SNAKE) | `MAX_SIZE`, `PI`, `BUFFER_LEN`          | 宏定义、常量          |
| **下划线前缀**                 | `_private_var`, `_internal_func`        | 表示"私有/内部"       |

### 10.2 命名示例

```c
// ✅ 好的命名——见名知意
int student_count = 30;
float average_score = 85.5;
char user_choice = 'y';
double circle_radius = 5.0;

// ✅ 宏常量的命名
#define MAX_BUFFER_SIZE 1024
#define PI 3.1415926
#define DEFAULT_PORT 8080
```

## 十一、C语言算术运算符与赋值运算符笔记

### 1. 算术运算符
算术运算符用于做数学计算。

#### 1.1 `+` 加法
```c
int a = 10;
int b = 3;
int c = a + b;   // c = 13
```

#### 1.2 `-` 减法
```c
int a = 10;
int b = 3;
int c = a - b;   // c = 7
```

#### 1.3 `*` 乘法
```c
int a = 10;
int b = 3;
int c = a * b;   // c = 30
```

#### 1.4 `/` 除法
```c
int a = 10;
int b = 3;
int c = a / b;   // c = 3
```

#### 1.5 `%` 取余
```c
int a = 10;
int b = 3;
int c = a % b;   // c = 1
```

### 2. 赋值运算符
赋值运算符用于把值存给变量。

#### 2.1 `=` 直接赋值
```c
int a = 10;
```

#### 2.2 `+=` 加后赋值
```c
int a = 10;
a += 5;   // a = a + 5，结果是 15
```

#### 2.3 `-=` 减后赋值
```c
int a = 10;
a -= 3;   // a = a - 3，结果是 7
```

#### 2.4 `*=` 乘后赋值
```c
int a = 10;
a *= 2;   // a = a * 2，结果是 20
```

#### 2.5 `/=` 除后赋值
```c
int a = 10;
a /= 2;   // a = a / 2，结果是 5
```

#### 2.6 `%=` 取余后赋值
```c
int a = 10;
a %= 3;   // a = a % 3，结果是 1
```
