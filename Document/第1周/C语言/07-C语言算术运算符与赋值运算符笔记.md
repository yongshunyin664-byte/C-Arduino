# C语言算术运算符与赋值运算符笔记

## 1. 算术运算符
算术运算符用于做数学计算。

### 1.1 `+` 加法
```c
int a = 10;
int b = 3;
int c = a + b;   // c = 13
```

### 1.2 `-` 减法
```c
int a = 10;
int b = 3;
int c = a - b;   // c = 7
```

### 1.3 `*` 乘法
```c
int a = 10;
int b = 3;
int c = a * b;   // c = 30
```

### 1.4 `/` 除法
```c
int a = 10;
int b = 3;
int c = a / b;   // c = 3
```

### 1.5 `%` 取余
```c
int a = 10;
int b = 3;
int c = a % b;   // c = 1
```

## 2. 赋值运算符
赋值运算符用于把值存给变量。

### 2.1 `=` 直接赋值
```c
int a = 10;
```

### 2.2 `+=` 加后赋值
```c
int a = 10;
a += 5;   // a = a + 5，结果是 15
```

### 2.3 `-=` 减后赋值
```c
int a = 10;
a -= 3;   // a = a - 3，结果是 7
```

### 2.4 `*=` 乘后赋值
```c
int a = 10;
a *= 2;   // a = a * 2，结果是 20
```

### 2.5 `/=` 除后赋值
```c
int a = 10;
a /= 2;   // a = a / 2，结果是 5
```

### 2.6 `%=` 取余后赋值
```c
int a = 10;
a %= 3;   // a = a % 3，结果是 1
```

## 3. 注释写法
### 3.1 单行注释
```c
// 这是单行注释
int a = 10;
```

### 3.2 多行注释
```c
/*
这是多行注释
可以写很多行
*/
```

## 4. 记忆口诀
- 算术运算符：负责“算”
- 赋值运算符：负责“存”
- 注释：给代码加说明
