# 一维数组：定义、赋值、下标访问

数组是**相同类型**数据的集合，在内存中**连续存放**，通过下标访问。

```c
int arr[5] = {10, 20, 30, 40, 50};  // 定义并初始化
arr[0] = 100;                        // 下标从 0 开始，修改第一个元素
int x = arr[2];                      // 访问第三个元素，x = 30
```

- `int arr[5]`：定义 5 个 int 元素的数组
- 下标范围：`0` ~ `4`（共 5 个）
- ⚠ 下标越界不会报错，但会破坏其他数据，是常见 bug
